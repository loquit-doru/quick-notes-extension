# Quick Notes - Privacy Policy

Last updated: January 21, 2026

## Overview
Quick Notes is committed to protecting your privacy. This extension is designed with privacy-first principles.

## Data Collection
**We do NOT collect:**
- Personal information
- Browsing history
- Note contents
- Usage analytics
- Any identifiable data

## Data Storage
- All notes are stored **locally** on your device using IndexedDB
- Notes never leave your browser unless you explicitly export them
- No cloud sync means no data transmission to external servers

## Third-Party Services
- **ExtensionPay**: Used only for processing card payments. We do not store payment details.
- **Base Network**: Used only for verifying crypto payments. Only transaction hashes are checked.

## Permissions Used
- **storage**: To save your notes locally
- **activeTab**: To capture page context (URL/title) when creating notes
- **tabs**: To get current tab information for context capture

## Contact
For privacy concerns, contact: quicknotes@example.com

## Changes
We may update this policy. Changes will be posted here.
